# LineArt Maps — Feature List

## GPS File Support
- GPX files (Strava, Garmin Connect, AllTrails, Komoot)
- TCX files (Garmin devices)
- FIT files (Garmin/Wahoo native format)
- Automatic metadata extraction (name, date, distance, duration)

## Customization
- 10 curated color presets (navy on cream, gold on black, coral on dark gray, and more)
- Custom route and background color pickers
- Line width slider (0.5px to 6px)
- Line style: solid, dashed, or dotted
- Route smoothing (Bezier curve interpolation, adjustable intensity)
- Portrait and landscape orientation

## Text Overlays
- Add unlimited text elements
- Drag-and-drop positioning
- 8 font choices: Arial, Montserrat, Raleway, Playfair Display, Georgia, Times New Roman, Dancing Script, Courier New
- Adjustable font size (8–72px)
- Custom text color
- Left, center, or right alignment
- Auto-populated activity metadata (name, date, distance, duration)

## Export
- PNG — high-resolution raster (10x scale)
- SVG — scalable vector graphics
- PDF — exact paper dimensions, print-ready
- Paper sizes: A5, A4, A3, A2, A1, A0

## Technical
- Pure HTML/CSS/JavaScript — no build system, no dependencies
- Works completely offline
- No account or login required
- Runs in any modern browser (Chrome, Firefox, Safari, Edge)
- Cross-platform: Windows, macOS, Linux
- Mobile-responsive interface
- Loading indicator for large GPS files
- One-click reset to start over
